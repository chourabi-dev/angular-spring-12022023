https://angular.io/guide/versions

1 - check if pc has nodejs
=> node -v
2 - check if pc has npm
=> npm -v
3 - install angular/cli
npm i -g @angular/cli@12
4 - check if angular successfuly installed
=> ng version
5 - create new project
=> ng new app


WARNING !!
if you have error while creating app please  open power shell as admin:
Set-ExecutionPolicy Unrestricted



6- start the development server
=> ng serve / npm start


7- install boostrap
=> npm i bootstrap


8 - create componenets:
=> ng g c comp_name


9 - we can create pipes using
=> ng g p pipe_name
